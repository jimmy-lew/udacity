# Running Instructions 


## API 


```bash
  
  cd api
  pip3 install -r requirements_pip.txt
  python3 main.py


```

## Web

Using npm

```bash

  cd web
  npm i 
  npm run dev


```

Using pnpm (prefered)

```bash

  cd web
  pnpm i
  pnpm dev

```

Alternatively you can use the deployed version [here](https://udacity-one.vercel.app)
